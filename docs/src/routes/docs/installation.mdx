# Installation

Choose your preferred installation method:

<InstallationMethodGrid onSelect={(method) => {
  console.log('method', method);
  setSelectedMethod(method);
  const element = document.getElementById(method);
  if (element) {
    const headerHeight = 80;
    const elementTop = element.offsetTop - headerHeight;
    window.scrollTo({
      top: elementTop,
      behavior: 'smooth'
    });
  }
}} />

## CLI

```sh
curl -fsSL https://tombi-toml.github.io/tombi/install.sh | sh
```

Install a specific version:

```sh
curl -fsSL https://tombi-toml.github.io/tombi/install.sh | sh -s -- --version 0.2.24
```
